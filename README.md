## Scripts

**⚠ サーバの起動コードが実装されるまでは、起動を伴うコマンドはいずれも動作しません。**

### 開発モードでの起動

```sh
npm run dev
```

現時点ではホットリロードは未実装です。

### ビルド

```sh
npm build
```

ビルド成果物の起動

```sh
npm start
```

### テストの実行

```sh
npm test
```
